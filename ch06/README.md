### 6.1 함수 추출하기 (Extract Function)

- "목적과 구현을 분리"하여 코드를 독립된 함수로 묶기

- 함수가 짧으면 캐싱하기가 쉬워서 컴파일러가 최적화하는데 유리할 때가 많음
