# CH06 기본적인 리팩터링

### 6.1 함수 추출하기 (Extract Function)

- "목적과 구현을 분리"하여 코드를 독립된 함수로 묶기
- 코드가 무슨 일 하는지 파악하기 어려운 경우 -> 함수로 추출 -> '무슨 일'에 맞게 이름 짓기
- 심지어 메서드 안에 구현 코드는 또 다른 메서드 하나만 호출하는 경우도 있음
- 함수가 짧으면 캐싱하기가 쉬워서 컴파일러가 최적화하는데 유리할 때가 많음
