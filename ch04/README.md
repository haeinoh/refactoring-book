- 리팩터링에는 견고한 테스트 스위트(test suite)가 뒷받침 되어야 함

### 4.1 자가 테스트 코드의 가치

- **모든 테스트를 완전히 자동화하고 그 결과까지 스스로 검사하게 만들자**

- 컴파일이 쉬워짐 -> 디버깅 시간 줄어듦 -> 생산성 상승

- **테스트 스위트는 강력한 버그 검출 도구로, 버그를 찾는데 걸리는 시간을 대폭 줄여준다**

- 테스트를 작성하기 가장 좋은 시점 -> 프로그래밍 시작 전 (**TDD**)

- 테스트 - 코딩 - 리팩터링
